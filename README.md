## ğŸ“ Usage

### Creating a Note
1. Navigate to the home page
2. Enter a title in the input field
3. Write your content in the textarea
4. Click "Create My Note" button

### Editing a Note
1. Click the "Edit" button on any note card
2. Modify the title or content
3. Click "Edit Note" to save changes

### Viewing a Note
1. Click the "View" button on any note card
2. Read the full note with formatted content
3. Use action buttons to edit, copy, share, or delete

### Searching Notes
1. Go to the Notes page
2. Type in the search bar
3. Notes are filtered in real-time

### Sharing a Note
1. Click the "Share" button on any note
2. On mobile/modern browsers: Native share menu opens
3. On desktop: Link is copied to clipboard
